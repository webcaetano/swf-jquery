# ![swfJQ Logo](http://i.imgur.com/Wed19Q4.png)

jQuery plugin that easily manipulate SWF based on SWFObject.

This awesome class exist since 2009. Now on Bower.

### Quickstart

Using [Bower](http://bower.io)
```
bower install swf-query --save
```
